# 알고리즘이란?

문제를 해결하기 위한 일련의 수학적 단계이다.

1. 문제를 해결을 위한 계획수립하기.(문제를 세분화하기)
2. 문제해결 패턴을 파악하기

## 문제해결의 단계

문제 이해하기-> 구체적인 예시 알아보기-> 문제를 세분화-> 문제를 해결-> 문제를 단순화-> 복습 후 재구성
